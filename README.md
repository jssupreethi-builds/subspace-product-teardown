# Subspace.money Product Teardown

## Overview

This repository contains my submission for the Subspace.money Product Intern Assignment.

The analysis is based on a hands-on evaluation of the platform, including:

- Home Dashboard
- Subscription Sharing Marketplace
- Explore Section
- Wallet
- Savings Dashboard
- Rental Marketplace
- Community Features

## Key Focus Areas

1. User Activation
2. Trust Building
3. Conversion Optimization
4. Retention
5. Marketplace Experience

## Deliverables

### Product Teardown Report
Detailed observations, identified problems, and prioritized recommendations.

### UI Improvement Mockups
Visual representations of proposed product improvements aligned with the existing Subspace.money design system.

### Supporting Screenshots
Screenshots captured during product exploration and evaluation.

## Recommendations Summary

| Priority | Recommendation |
|-----------|---------------|
P1	Cold start onboarding fix
P1	Intent-based discovery
P1	Trust Score on sharers
P1	Surface savings everywhere
P2	Refocus rental marketplace


## Product Thinking Approach

Recommendations were prioritized based on expected impact across key funnel stages:

- Activation
- Conversion
- Retention

while keeping implementation effort realistic for a startup product team.
